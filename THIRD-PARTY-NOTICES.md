# Third-Party Notices

Logos is distributed under the [Apache License 2.0](LICENSE) (NFR-CR-01).

The distributed `logos` binary statically links the third-party Rust crates
and C libraries below (NFR-CR-02). Each is used under the license indicated;
the full license texts are available from the linked repositories. The
bundled SQLite amalgamation (via `rusqlite`'s `bundled` feature) is public
domain. Tree-sitter grammars are compiled in as generated C parse tables
under their crates' licenses listed below.

The `--features ui` release binary additionally embeds the web dashboard SPA
([CR-049](docs/requests/CR-049-decouple-web-ui-into-embedded-spa.md),
FR-UI-22): the **bundled JavaScript** section below covers the production npm
dependency closure compiled into that embedded bundle. The build-time-only
toolchain (Vite, TypeScript, …) does not ship and is excluded.

This file is generated by `scripts/gen-third-party-notices.py`; regenerate it
whenever `Cargo.lock` or `web/ui/package-lock.json` changes the shipped
dependency set.
